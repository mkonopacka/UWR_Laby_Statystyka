Zadanie 1

Wielkość \(P(X \leq 3)\) jest funkcją parametru \(p\): \[P(p) = {5 \choose 3}p^3(1-p)^{1-3} + {5 \choose 4}p^4(1-p)^{1-4} + {5 \choose 5}p^5(1-p)^{1-5}\] stąd estymator nawiększej wiarogodności \(P\) otrzymamy wyznaczając ENW \(p\) (dla tego rozkładu jest nim proporcja sukcesów) i przekazując otrzymaną wartość do funkcji \(P(p)\).

Zadanie 2

Estymatorem największej wiarogodności parametru \(\lambda\) jest średnia arytmetyczna. W celu wyznaczenia ENW \(P(X = x)\), korzystamy z tego samego sposobu co w zadaniu 1.

  1. Wariancja jest niezależnie od \(\lambda\) praktycznie zerowa.
  2. Obciążenie i bezwzględna wartość średniego błędu kwadratowego rosną wraz ze wzrostem \(\lambda\), ale nie różnią się znacząco dla różnych wartości \(x\). Przeprowadziłam dodatkowe eksperymenty i sprawdziłam jak wygląda zależność obciążenia i MSE od \(\lambda\).

Zadanie 4

Gdy \(\beta = 1\), to: funkcja masy prawdopodobieństwa jest postaci \(P(x|\theta) = \theta x^{\theta - 1}\), informacja Fishera jest funkcją parametru \(\theta\): \(I(\theta) = \frac{1}{\theta^2}\), a estymator największej wiarogodności \(\theta\) można wyrazić zwartym wzorem: \(\hat{\theta} = \frac{-n}{\sum{ln(x_i)}}\). Z Tw.1 \(\hat{I(\theta)} = I(\hat{\theta})\).

mle_beta1 <- function(X){
  n = length(X)
  return(-n/sum(log(X)))
}

fisher <- function(theta){return(1/theta^2)}

W pierwszej cześci zadania estymowałam informacje Fishera dla wartości \(\theta\) uzyskanych dla 10000 różnych prób:

zadanie4_1 <- function(th, n = 50, iter = 10000, seed = 1){
  set.seed(seed)
  X <- numeric(0) # tu będą dołączane wyniki każdego z eksperymentów, czyli estymowane wartości Informacji Fishera
  for (i in 1:iter){
    sample <- rbeta(n, th, 1)
    mleX <- mle_beta1(sample)
    X[i] <- fisher(mleX)
  results <- data.frame(n = n,
                        theta = th,
                        MSE = do.call(function(X){mean((X-fisher(th))^2)}, list(X)), 
                        bias = do.call(function(X){mean(X-fisher(th))}, list(X)),
                        var = do.call(var, list(X)),
                        mean_I = mean(X),
                        real_I = fisher(th))
  }
  # w wyniku zwracam podsumowanie (results) oraz wektor obliczonych informacji Fishera, żeby rysować z niego boxplota
  return(list(results, X))
}

W dalszej części zadania za zapamiętaną wartość estymatora \(\hat{I(\theta)}\) przyjęłam średnią arytmetyczną wyników. Powinno to mieć tym mniejsze znaczenie, im większa będzie wartość \(\theta\) - ze względu na mniejszy rozrzut estymatora. Tym razem wygeneruję inne próby losowe (w kodzie skorzystałam z innej wartości set.seed())

zadanie4_2 <- function(th, mean_I, n = 50, iter = 10000, seed = 38){
  # def. zmienna Y
  Y <- function(n, th, th_est, mean_I){
    return((th_est-th)*sqrt(n*mean_I))
  }
}
