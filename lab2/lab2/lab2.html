Zadanie 4

Gdy \(\beta = 1\), to: funkcja masy prawdopodobiestwa jest postaci \(P(x|\theta) = \theta x^{\theta - 1}\), informacja Fishera jest funkcj parametru \(\theta\): \(I(\theta) = \frac{1}{\theta^2}\), a estymator najwikszej wiarogodnoci \(\theta\) mona wyrazi zwartym wzorem: \(\hat{\theta} = \frac{-n}{\sum{ln(x_i)}}\). Z Tw.1 \(\hat{I(\theta)} = I(\hat{\theta})\).

mle_beta1 <- function(X){
  n = length(X)
  return(-n/sum(log(X)))
}

fisher <- function(theta){return(1/theta^2)}

W pierwszej czeci zadania estymowaam informacje Fishera dla wartoci \(\theta\) uzyskanych dla 10000 rnych prb:

zadanie4_1 <- function(th, n = 50, iter = 10000, seed = 1){
  set.seed(seed)
  X <- numeric(0) # tu bd doczane wyniki kadego z eksperymentw, czyli estymowane wartoci Informacji Fishera
  for (i in 1:iter){
    sample <- rbeta(n, th, 1)
    mleX <- mle_beta1(sample)
    X[i] <- fisher(mleX)
  results <- data.frame(n = n,
                        theta = th,
                        MSE = do.call(function(X){mean((X-fisher(th))^2)}, list(X)), 
                        bias = do.call(function(X){mean(X-fisher(th))}, list(X)),
                        var = do.call(var, list(X)),
                        mean_I = mean(X),
                        real_I = fisher(th))
  }
  # w wyniku zwracam podsumowanie (results) oraz wektor obliczonych informacji Fishera, eby rysowa z niego boxplota
  return(list(results, X))
}
